How it works

Linear Search — O(n)

Iterates through all … words from start to finish. For every word it checks whether the word begins with the typed prefix. In the worst case it must inspect every single word.

Binary Search — O(log n)

The word list is kept sorted. Two binary searches pinpoint the exact range of matching words in O(log n) comparisons — a classic divide-and-conquer strategy. Only the boundary indices are computed; no full scan is needed.
